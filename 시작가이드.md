# 🚀 시작 가이드

## 1단계: PostgreSQL 설치 및 DB 생성

### 방법 A — Docker 사용 (권장)
```bash
# Docker가 설치되어 있다면:
docker run --name haniwon-db -e POSTGRES_PASSWORD=password -e POSTGRES_DB=haniwon_db -p 5432:5432 -d postgres
```

### 방법 B — PostgreSQL 직접 설치
1. https://www.postgresql.org/download/ 에서 Windows용 설치
2. pgAdmin 또는 psql에서 DB 생성:
   ```sql
   CREATE DATABASE haniwon_db;
   ```

---

## 2단계: 환경변수 설정

`.env.local` 파일에서 DB 정보 수정:
```
DATABASE_URL="postgresql://postgres:password@localhost:5432/haniwon_db"
```

---

## 3단계: DB 스키마 적용 + 시드 데이터

```bash
# 스키마 적용
npm run db:push

# 시드 데이터 생성 (샘플 한의원, 품목, 관리자 계정)
npx ts-node --compiler-options '{"module":"CommonJS"}' prisma/seed.ts
```

---

## 4단계: 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

## 주요 명령어

| 명령어              | 설명                     |
|---------------------|--------------------------|
| `npm run dev`       | 개발 서버 시작            |

| `npm run db:studio` | DB 관리 UI (Prisma Studio)|
| `npm run db:migrate`| 마이그레이션 생성         |
| `npm run build`     | 프로덕션 빌드             |

## 파일 구조

```
haniwon-order-system/
├── app/
│   ├── (auth)/login/       ← 로그인 페이지
│   ├── (dashboard)/        ← 인증 필요 페이지
│   │   ├── dashboard/      ← 대시보드

│   └── api/                ← REST API
├── components/             ← UI 컴포넌트
├── lib/                    ← DB, Auth, Utils
├── prisma/                 ← DB 스키마 + 시드
└── types/                  ← TypeScript 타입
