# 03. DB 설계 (ERD 개요)

## 테이블 관계도

```
users (사용자)
  └─< clinics (한의원)          ← sales_rep_id FK
  └─< orders (주문)             ← created_by FK

clinics (한의원)
  └─< delivery_addresses        ← 납품지 복수
  └─< orders (주문)
  └─< invoices (청구서)
  └─< clinic_prices (계약단가)

items (품목)
  └─< order_items               ← 주문 품목
  └─< delivery_items            ← 납품 품목
  └─< clinic_prices             ← 한의원별 단가

orders (주문)
  └─< order_items               ← 주문 품목 상세
  └── deliveries (납품)         ← 1주문:1납품 (기본)

deliveries (납품)
  └─< delivery_items            ← 실납품 품목

invoices (청구서)
  └─< invoice_orders            ← 청구서 포함 주문
  └─< payments (입금내역)
```

---

## SQL 테이블 정의 (PostgreSQL 기준)

### users
```sql
CREATE TABLE users (
  user_id     UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name        VARCHAR(50)  NOT NULL,
  email       VARCHAR(100) UNIQUE NOT NULL,
  password    VARCHAR(255) NOT NULL,
  role        VARCHAR(20)  NOT NULL CHECK (role IN ('admin','sales','clinic')),
  phone       VARCHAR(20),
  created_at  TIMESTAMP DEFAULT NOW(),
  updated_at  TIMESTAMP DEFAULT NOW()
);
```

### clinics
```sql
CREATE TABLE clinics (
  clinic_id       UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  clinic_name     VARCHAR(100) NOT NULL,
  ceo_name        VARCHAR(50),
  business_no     VARCHAR(20) UNIQUE,
  address         TEXT,
  phone           VARCHAR(20),
  email           VARCHAR(100),
  payment_type    VARCHAR(30) DEFAULT '후불_월마감',
  credit_limit    NUMERIC(15,2) DEFAULT 0,
  sales_rep_id    UUID REFERENCES users(user_id),
  start_date      DATE,
  status          VARCHAR(20) DEFAULT '활성',
  created_at      TIMESTAMP DEFAULT NOW(),
  updated_at      TIMESTAMP DEFAULT NOW()
);
```

### items
```sql
CREATE TABLE items (
  item_id           UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  item_code         VARCHAR(50) UNIQUE,
  item_name         VARCHAR(200) NOT NULL,
  spec              VARCHAR(100),
  unit              VARCHAR(20),
  category          VARCHAR(30),
  supply_price      NUMERIC(15,2) NOT NULL,
  list_price        NUMERIC(15,2),
  is_medical_device BOOLEAN DEFAULT FALSE,
  license_no        VARCHAR(100),
  device_grade      SMALLINT,
  license_expiry    DATE,
  stock_qty         INTEGER DEFAULT 0,
  manufacturer      VARCHAR(100),
  supplier          VARCHAR(100),
  status            VARCHAR(20) DEFAULT '판매중',
  created_at        TIMESTAMP DEFAULT NOW(),
  updated_at        TIMESTAMP DEFAULT NOW()
);
```

### orders
```sql
CREATE TABLE orders (
  order_id                UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_no                VARCHAR(50) UNIQUE,
  order_date              TIMESTAMP DEFAULT NOW(),
  clinic_id               UUID REFERENCES clinics(clinic_id),
  delivery_address        TEXT,
  order_channel           VARCHAR(30),
  requested_delivery_date DATE,
  total_amount            NUMERIC(15,2),
  status                  VARCHAR(30) DEFAULT '접수',
  memo                    TEXT,
  sales_rep_id            UUID REFERENCES users(user_id),
  created_by              UUID REFERENCES users(user_id),
  created_at              TIMESTAMP DEFAULT NOW(),
  updated_at              TIMESTAMP DEFAULT NOW()
);
```

### order_items
```sql
CREATE TABLE order_items (
  order_item_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id      UUID REFERENCES orders(order_id) ON DELETE CASCADE,
  item_id       UUID REFERENCES items(item_id),
  qty           INTEGER NOT NULL,
  unit_price    NUMERIC(15,2) NOT NULL,
  amount        NUMERIC(15,2) GENERATED ALWAYS AS (qty * unit_price) STORED,
  is_returned   BOOLEAN DEFAULT FALSE
);
```

### deliveries
```sql
CREATE TABLE deliveries (
  delivery_id    UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  delivery_no    VARCHAR(50) UNIQUE,
  order_id       UUID REFERENCES orders(order_id),
  delivery_date  DATE,
  delivered_by   VARCHAR(50),
  receiver_name  VARCHAR(50),
  signature_url  TEXT,
  photo_url      TEXT,
  statement_url  TEXT,
  created_at     TIMESTAMP DEFAULT NOW()
);
```

### delivery_items
```sql
CREATE TABLE delivery_items (
  delivery_item_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  delivery_id      UUID REFERENCES deliveries(delivery_id) ON DELETE CASCADE,
  item_id          UUID REFERENCES items(item_id),
  ordered_qty      INTEGER,
  delivered_qty    INTEGER,
  is_return        BOOLEAN DEFAULT FALSE
);
```

### invoices
```sql
CREATE TABLE invoices (
  invoice_id      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  invoice_no      VARCHAR(50) UNIQUE,
  clinic_id       UUID REFERENCES clinics(clinic_id),
  billing_period  VARCHAR(7),   -- 예: '2026-05'
  issue_date      DATE,
  supply_amount   NUMERIC(15,2),
  vat_amount      NUMERIC(15,2),
  total_amount    NUMERIC(15,2),
  tax_invoice_no  VARCHAR(100),
  status          VARCHAR(20) DEFAULT '발행',
  due_date        DATE,
  created_at      TIMESTAMP DEFAULT NOW(),
  updated_at      TIMESTAMP DEFAULT NOW()
);
```

### payments
```sql
CREATE TABLE payments (
  payment_id   UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  invoice_id   UUID REFERENCES invoices(invoice_id),
  paid_date    DATE,
  paid_amount  NUMERIC(15,2),
  pay_method   VARCHAR(20),
  memo         TEXT,
  created_at   TIMESTAMP DEFAULT NOW()
);
```
