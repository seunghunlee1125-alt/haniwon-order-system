# 01. 도메인 정의

## 1. 기관(고객) 관리

### 필드 목록
| 필드명 | 타입 | 설명 |
|--------|------|------|
| clinic_id | UUID | 한의원 고유 ID |
| clinic_name | STRING | 원명 |
| ceo_name | STRING | 대표원장명 |
| business_no | STRING | 사업자번호 (000-00-00000) |
| address | STRING | 사업장 주소 |
| delivery_address | STRING | 납품지 주소 (본원/분원) |
| phone | STRING | 대표 연락처 |
| email | STRING | 이메일 |
| payment_type | ENUM | 결제방식: 후불_월마감 / 선불 / 카드 |
| credit_limit | NUMBER | 신용 한도 (원) |
| sales_rep_id | UUID | 담당 영업사원 ID |
| start_date | DATE | 거래 시작일 |
| status | ENUM | 상태: 활성 / 휴면 / 거래중단 |

---

## 2. 상품(품목) 관리

### 필드 목록
| 필드명 | 타입 | 설명 |
|--------|------|------|
| item_id | UUID | 품목 고유 ID |
| item_code | STRING | 품목 코드 |
| item_name | STRING | 품명 |
| spec | STRING | 규격 (예: 100매/박스) |
| unit | STRING | 단위 (box, ea, set, ...) |
| category | ENUM | 소모품 / 의료기기 / 한약재 / 기타 |
| supply_price | NUMBER | 공급가 (원) |
| list_price | NUMBER | 권장소비자가 (원) |
| is_medical_device | BOOLEAN | 의료기기 여부 |
| license_no | STRING | 의료기기 허가번호 (해당시) |
| device_grade | ENUM | 의료기기 등급: 1~4 (해당시) |
| license_expiry | DATE | 허가 유효기간 |
| stock_qty | NUMBER | 재고 수량 |
| manufacturer | STRING | 제조사 |
| supplier | STRING | 공급사 |
| status | ENUM | 상태: 판매중 / 단종 / 일시품절 |

---

## 3. 주문 관리

### 필드 목록
| 필드명 | 타입 | 설명 |
|--------|------|------|
| order_id | UUID | 주문 고유 ID |
| order_no | STRING | 주문번호 (예: ORD-20260526-001) |
| order_date | DATETIME | 주문일시 |
| clinic_id | UUID | 주문 한의원 |
| delivery_address | STRING | 납품지 주소 |
| order_channel | ENUM | 앱 / 전화 / 영업사원대리입력 / 웹 |
| requested_delivery_date | DATE | 요청 납품일 |
| items | ARRAY | 주문 품목 리스트 (하단 참고) |
| total_amount | NUMBER | 합계 금액 |
| status | ENUM | 접수→확인→출고준비→출고→납품완료→정산 |
| memo | TEXT | 특이사항 메모 |
| sales_rep_id | UUID | 담당 영업사원 |

### 주문 품목 (order_items)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| order_item_id | UUID | |
| order_id | UUID | 주문 ID (FK) |
| item_id | UUID | 품목 ID (FK) |
| qty | NUMBER | 주문 수량 |
| unit_price | NUMBER | 적용 단가 |
| amount | NUMBER | 금액 (qty × unit_price) |
| is_returned | BOOLEAN | 반품 여부 |

### 주문 상태 흐름
```
접수 → 확인 → 출고준비 → 출고 → 납품완료 → 정산완료
         ↓
       취소/반려
```

## 4. 납품 관리

### 필드 목록
| 필드명 | 타입 | 설명 |
|--------|------|------|

| return_items | ARRAY | 반품 수거 품목 |
| statement_url | STRING | 거래명세서 PDF URL |

## 5. 정산·청구 관리

### 청구서 (invoice)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| invoice_id | UUID | 청구서 ID |
| invoice_no | STRING | 청구서 번호 |
| clinic_id | UUID | 대상 한의원 |
| billing_period | STRING | 청구 기간 (예: 2026-05) |
| issue_date | DATE | 발행일 |
| supply_amount | NUMBER | 공급가액 |
| vat_amount | NUMBER | 부가세 (10%) |
| total_amount | NUMBER | 합계 |
| tax_invoice_no | STRING | 세금계산서 번호 |
| status | ENUM | 발행 / 미수 / 입금완료 / 연체 |
| due_date | DATE | 납기일 |

### 입금 내역 (payments)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| payment_id | UUID | |

| paid_date | DATE | 입금일 |
| paid_amount | NUMBER | 입금액 |
| pay_method | ENUM | 현금 / 계좌이체 / 카드 |
| memo | TEXT | 비고 |
