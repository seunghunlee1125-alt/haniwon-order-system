# 04. 화면 목록 (Screen List)

## 관리자/영업사원 화면

### 공통
| 화면 | URL | 설명 |
|------|-----|------|
| 로그인 | /login | 이메일+비밀번호 |
| 대시보드 | /dashboard | 오늘 주문수, 미수금, 납품예정 요약 |

### 기관(한의원) 관리
| 화면 | URL | 설명 |
|------|-----|------|
| 한의원 목록 | /clinics | 검색, 상태 필터 |
| 한의원 등록 | /clinics/new | 신규 등록 폼 |
| 한의원 상세 | /clinics/:id | 기본정보 + 거래이력 탭 |
| 한의원 수정 | /clinics/:id/edit | |

### 상품(품목) 관리
| 화면 | URL | 설명 |
|------|-----|------|
| 품목 목록 | /items | 카테고리 필터, 검색 |
| 품목 등록 | /items/new | |
| 품목 상세/수정 | /items/:id | |

### 주문 관리
| 화면 | URL | 설명 |
|------|-----|------|
| 주문 목록 | /orders | 상태별 탭, 날짜 필터 |
| 주문 접수 | /orders/new | 한의원 선택 → 품목 추가 |
| 주문 상세 | /orders/:id | 상태 변경 버튼 포함 |

### 납품 관리
| 화면 | URL | 설명 |
|------|-----|------|
| 납품 목록 | /deliveries | 날짜별 납품 예정/완료 |
| 납품 처리 | /deliveries/:id | 실납품 수량 입력, 서명 |
| 거래명세서 | /deliveries/:id/statement | PDF 미리보기 + 출력 |

### 정산·청구 관리
| 화면 | URL | 설명 |
|------|-----|------|
| 청구서 목록 | /invoices | 상태 필터(미수/완료/연체) |
| 청구서 상세 | /invoices/:id | 포함 주문 목록, 입금 등록 |
| 미수금 현황 | /invoices/receivables | 한의원별 미수금 합계 |
| 매출 리포트 | /reports | 월별/한의원별/품목별 차트 |

## 한의원(원장) 화면 (2단계)

| 화면 | URL | 설명 |
|------|-----|------|
| 주문하기 | /order | 품목 카탈로그 → 장바구니 → 주문 |
| 주문 내역 | /my/orders | 내 주문 목록 및 상태 |
| 납품 확인 | /my/deliveries | 납품서 조회, 서명 |
| 청구서 조회 | /my/invoices | 월별 청구서 및 납부 현황 |

---

## 화면별 주요 컴포넌트

```
공통 레이아웃
├── 사이드바 네비게이션
├── 상단 헤더 (사용자 정보, 알림)
└── 메인 컨텐츠 영역

주문 접수 폼
├── 한의원 검색 선택 (autocomplete)
├── 품목 추가 (품목 검색 → 수량 입력)
├── 품목 리스트 테이블 (수량/단가/금액 자동계산)
├── 합계 금액 표시
└── 납품 요청일 / 메모 입력

거래명세서 PDF
├── 공급자 정보 (회사명, 사업자번호)
├── 공급받는자 정보 (한의원명, 사업자번호)
├── 품목 테이블 (품명, 규격, 수량, 단가, 금액)
├── 합계 (공급가액, 부가세, 총액)
└── 납품일, 납품 담당자, 수령자 서명란
```
